# 6651-Final-Project
Final Project for CSCI 6651

## About
This is a multi-game application that allows the user to play 3 fun classic games: Hangman, Snake, and Checkers

## Install Dependencies
```
pip install -r requirements.txt
```

## Usage
```
python3 ./main.py
```
